How does JavaScript work?

JavaScript is a scripting language for the web. It is an interpreted language, This means it does not require to compile the JavaScript source code before sending it to the browser.JavaScript has native asynchronous mechanisms that let it rect to events.JavaScript is also a dynamically typed language, unlike C and C++. Javascript is a single threaded and single concurrent programming language.

What does JavaScript event loop do?

JavaScript code runs single threaded. Only one line of code can read at a single time. The event loop works by making a request to some inside or outside event provider, then calls the relevant event handler.it constantly checks whether or not the call stack is empty. LocalStorage vs SessionStorage.

localStorage persists until explicitly deleted. Changes made are saved and available for all current and future visits to the site.
sessionStorage, changes are only available per tab. Changes made are saved and available for the current page in that tab until it is closed. Once it is closed, the stored data is deleted.
localStorage has a different expiration time, sessionStorage will only be accessible while and by the window that created it is open.
localStorage - use for long term use.
